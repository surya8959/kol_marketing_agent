# 🎙️ 播客内容再利用 Agent 使用指南

## 功能简介

播客内容再利用 Agent（Podcast Content Repurpose Agent）是 VANZO KOL Agent 系统的新增功能，旨在帮助播客创作者**一键将播客文稿转化为多平台的宣传物料**，最大化内容价值。

## 核心价值

| 问题 | 解决方案 |
|------|---------|
| 播客内容难以二次利用 | 自动转化为多种平台适配的内容 |
| 手动改写效率低 | AI 批量生成，1 分钟产出 7+ 平台内容 |
| 不同平台风格各异 | 针对平台特色定制文案 |
| 缺乏视频脚本 | 自动生成 TikTok/播客切片脚本 |

## 支持的平台

### 📕 小红书
- **内容类型**：图文笔记
- **风格**：接地气、生活化、实用性强
- **特点**：emoji 丰富、段落分明、结尾互动

- **内容类型**：短推文
- **风格**：简洁有力、观点鲜明
- **特点**：黄金首句、线程格式

- **内容类型**：专业文章
- **风格**：深度专业、行业洞察
- **特点**：结构清晰、职业价值
- **字数限制**：3000 字

- **内容类型**：图片配文
- **风格**：视觉优先、生活方式
- **特点**：配合图片、美观精致
- **字数限制**：2200 字

- **内容类型**：短视频脚本
- **风格**：节奏快、情绪强、有反转

- **时长**：30-60 秒

### 🎙️ 播客切片
- **内容类型**：短视频切片脚本
- **风格**：金句优先、观点鲜明
- **特点**：悬念引导、平台适配
- **时长**：30-90 秒

### 📧 Newsletter
- **内容类型**：邮件摘要
- **风格**：深度总结、价值导向
- **特点**：结构清晰、要点提取
- **字数限制**：2000 字

## 使用流程

### 1️⃣ 输入播客信息

在页面切换到"播客推广"模式，填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| **播客标题** | 当期节目的标题 | "EP12: 深度对话 AI 时代的个人成长" |
| **节目名称** | 播客节目的名称 | "科技观察播客" |
| **主题/话题** | 节目涉及的话题 | "AI、创业、职业发展" |
| **播客文稿** | 完整的播客文案内容 | 粘贴完整文稿 |
| **目标平台** | 选择要生成的平台 | 多选 |

### 2️⃣ 一键生成

点击"✨ 一键生成宣传物料"按钮，AI 开始工作：

```
输入播客文稿
    ↓
分析核心内容
    ↓
提取关键信息（关键词、金句、章节）
    ↓
并行生成多平台内容
    ↓
输出可复制的文案
```

### 3️⃣ 复制使用

- 点击平台标签切换查看不同平台的内容
- 点击"📋 复制"按钮一键复制文案
- 直接粘贴到对应平台发布

## 技术实现

### Agent 架构

```
┌─────────────────────────────────────────────────────┐
│        Podcast Repurpose Agent                       │
├─────────────────────────────────────────────────────┤
│                                                     │
│  输入：播客文稿 + 元信息                             │
│        ↓                                           │
│  ┌─────────────────────────────────────────────┐   │
│  │          内容分析引擎                        │   │
│  │  • 关键词提取                                 │   │
│  │  • 金句识别                                   │   │
│  │  • 章节分割                                   │   │
│  │  • 情感分析                                   │   │
│  └─────────────────────────────────────────────┘   │
│        ↓                                           │
│  ┌─────────────────────────────────────────────┐   │
│  │        多平台生成引擎                        │   │
│  │  小红书 │ Twitter │ LinkedIn │ Instagram    │   │
│  │  TikTok │ 播客切片 │ Newsletter            │   │
│  └─────────────────────────────────────────────┘   │
│        ↓                                           │
│  输出：平台适配内容                                 │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 核心算法

**关键词提取**：
- TF-IDF 算法
- 停用词过滤
- 频率统计

**金句识别**：
- 感叹号/疑问句识别
- 长度筛选（10-80 字）
- 情感词匹配

**智能分段**：
- 按段落分割
- 保持语义完整
- 控制字数限制

## 常见问题

### Q1: 播客文稿支持多长？
**A**: 理论上无限制，但建议 500-5000 字，效果最佳。

### Q2: 生成的内容可以直接发布吗？
**A**: 建议人工审核后发布，AI 生成内容可能需要微调。

### Q3: 支持多语言吗？
**A**: 当前版本主要支持中文，英文功能开发中。

### Q4: 如何获得更高质量的内容？
**A**:
- 提供结构清晰的文稿
- 明确主题和话题
- 确保文稿质量

### Q5: 可以批量处理多期播客吗？
**A**: 当前版本仅支持单期处理，批量处理功能开发中。
