# 快速开始：图片 Prompt 生成器

## 🚀 30 秒上手

### 1. 生成图文种草物料
在应用中：
- 切换到"播客推广"模式
- 添加播客文案内容
- 生成深度长文
- 点击"📕 图文种草"的"生成"按钮

### 2. 查看生成的金句卡片
系统会自动：
- 提取 3-4 句金句
- 为每句生成 3 种风格的 AI 图片 Prompt
- 提供设计建议（布局、色彩、情调）

### 3. 复制 Prompt 使用
- 点击"📋 复制"按钮
- 粘贴到 Midjourney/Stable Diffusion/DALL-E
- 获得专业级图片

---

## 📋 三种风格速查

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **极简留白** | 大量留白、单色调、高级感 | 专业内容、品牌形象 |
| **杂志时尚** | 非对称、层次分明、大胆色彩 | 年轻化、时尚生活方式 |
| **渐变高级** | 居中极简、微妙渐变、质感 | 高端服务、商业内容 |

---

## 🎨 Prompt 规则速查

### ✅ 必须包含
```
- generous negative space (大量留白)
- minimalist composition (极简构图)
- Swiss/Japanese design style (设计流派)
```

### ❌ 必须避免
```
--no clutter, shapes, gradients, complex patterns, watermark, signature, text
```

## 💡 使用技巧

### 技巧 1：风格匹配
- 专业观点 → 极简留白
- 年轻化内容 → 杂志时尚

### 技巧 2：平台适配
- 小红书：杂志时尚、渐变高级
- 朋友圈：极简留白
- LinkedIn：极简留白

### 技巧 3：批量生成
- 同一金句生成 3 种风格
- A/B 测试哪个效果更好
- 根据数据优化选择

## 🔧 常见问题

**Q: 生成效果不理想怎么办？**
A:
1. 检查是否完整复制了 Prompt
2. 确认 AI 工具的设置
3. 可以微调 Prompt 中的某些关键词

**Q: 能否自定义风格？**
A: 可以！参考代码中的风格模板，修改 `image-prompt-generator.js`

**Q: 支持哪些 AI 工具？**
A: Midjourney、Stable Diffusion、DALL-E、Firefly 等

**Q: Prompt 太长怎么办？**
A: 已经优化过，主流 AI 工具都支持。如果仍有问题，可以删除部分质量词
